# Document API example (Streamlit)

```
Streamlit/
├── .streamlit/
│   ├── config.toml
│   └── sectets.toml # UPSTAGE_API_KEY 삽입
├── main.py # 메인 코드
├── requirements.txt
└── README.md
```

## 사용 예시시
1. Python 환경 구축.
```
pip install -r requirements.txt
```
2. Streamlit 실행 (`cd Streamlit` 먼저하기)
```
streamlit run main.py
```
